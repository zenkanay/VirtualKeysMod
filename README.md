# VirtualKeysMod

Minecraftでホットキーに割り当てるためのキーが足りなくなったプレイヤーや、複雑なキーバインドを整理したい方のために，仮想キー（Virtual Key）」を追加する Fabric Mod です。

## 機能

[こちら](https://youtu.be/So0EKyhCUyM?si=BwmiHPsH7gl8l_Z0)の動画をご覧ください．

## 動作環境

*   **Minecraft**: `1.21.11`
*   **Mod Loader**: `Fabric` (Fabric Loader `0.19.3` 以上)
*   **前提Mod**: `Fabric API`
