# VirtualKeys (Forge Version)

A client-side Forge mod that adds an on-screen virtual key panel (GUI overlay) to Minecraft, designed for players who run out of physical hotkeys or want to organize complex keybindings cleanly.

## Features

Please watch [this video](https://youtu.be/So0EKyhCUyM?si=BwmiHPsH7gl8l_Z0) to see how it works.

## Requirements

*   **Minecraft**: `1.20.1`
*   **Mod Loader**: `Forge` (Forge `47.1.0` or higher)
*   **Dependencies**:
    *   *(Highly Recommended)* `mafglib` (to bind virtual keys inside Forge Tweakeroo, Litematica, etc.)
